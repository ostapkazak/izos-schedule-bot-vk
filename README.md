# Бот с расписанием занятий и экзаменов студентов кафедры ИЗОС

## **Описание команд**

**_Установка номера группы для беседы_**

Формат команды: `!группа [номер]`

Задаёт номер группы для текущей беседы

**_Расписание на сегодня_**

Формат команды: `!сегодня`

Выводит список занятий на текущие сутки с учётом четности недели для установленной группы

**_Расписание на неделю_**

Формат команды: `!расписание`

Выводит полное расписание на неделю для установленной группы

**_Список экзаменов_**

Формат команды: `!экзамены`

Выводит список актуальных экзаменов для установленной группы

**_Список команд_**

Формат команды: `!помощь`

Выводит список доступных команд бота с описанием

**_Флаг рассылки расписания_**

Формат команды: `!рассылка`

Включает/выключает ежедневную рассылку расписания

## **Виды рассылок**

**_Рассылка расписания_**

Ежедневно высылает расписание на текущие сутки в беседы, где включена рассылка

**_Рассылка экзаменов_**

Высылает напоминание о сегодняшнем экзамене в беседы, где таковой имеется

## **Настройка расписания**

Изменение расписания занятий и экзаменов производится старостой группы через корпоративный сайт
